# Markdown 문법
## 1. 헤더
```
# 가장 큰 제목
## 두번째 큰 제목
### 세번째 큰 제목
#### 네번째 큰 제목
##### 다섯번째 큰 제목
###### 여섯번째 큰 제목
```
## 2. 목록
### 순서가 있는 목록은 숫자와 점을 사용합니다.

```
1. 첫번째
2. 두번째
3. 세번째
```

### 순서가 없는 목록(*,+,-)

```
* 빨강
    * 녹색
        * 파랑

+ 빨강
    + 녹색
        + 파랑

- 빨강
    - 녹색
        - 파랑
```

혼합하여 사용도 가능합니다.

```
* 1단계
    - 2단계
        + 3단계
            + 4단계
```
